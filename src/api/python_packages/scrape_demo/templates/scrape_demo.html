Scrape Laboratory

Try POST /scrape — Mercury-backed article extraction with optional Redis cache

What this endpoint does

  • POST /scrape accepts JSON with a required url string.
  • Optional contentType (string) is forwarded to mercury-parser (default on the server is markdown).
  • Optional headers must be a string in the format mercury-parser expects (URL-encoded header pairs).
  • Google News links are resolved to the underlying article URL before parsing.
  • URLs whose path looks like a direct file download (e.g. .pdf) are rejected with HTTP 400.
  • Successful responses are cached in Redis when available (same key as the API uses).
