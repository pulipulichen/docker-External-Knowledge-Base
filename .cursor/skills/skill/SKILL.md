# Skill

## Instructions

When creating or editing Cursor skills for this project:

1. Write skill metadata, headings, instructions, examples, and supporting content in English.
2. Keep the skill concise and specific to the workflow it teaches.
3. Use clear trigger terms in the `description` so the skill can be discovered reliably.
4. Store project skills under `.cursor/skills/<skill-name>/SKILL.md`.
